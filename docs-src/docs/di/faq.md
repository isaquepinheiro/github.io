## Posso usar tags para classes?

Hoje, o registro padrão de classes usa `T.ClassName` como chave. Tags são usadas principalmente para interfaces (e para injetar um injector filho com `AddInject`).

---

## Can I tag class registrations?

Currently, class registration uses `T.ClassName` as the key. Tags are mainly used for interfaces (and when adding a child injector via `AddInject`).
