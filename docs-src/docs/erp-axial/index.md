**ERP-Axial** é o host app Flutter que materializa o framework **Coreflow / Axial** como um ERP multi-abas voltado a cadastros administrativos, fiscais e operacionais migrados do Delphi legado.

## Onde começar

- [Introdução](introduction.md) — o que é, para quem é e qual problema resolve
- [Quickstart](getting-started/quickstart.md) — instalar, configurar e rodar a primeira vez
- [Configuração](reference/configuration.md) — variáveis `--dart-define`, perfis de execução, backend REST
- [Erros comuns](troubleshooting/common-errors.md) — sintomas, causas e ações

## Escopo

Esta documentação cobre o **uso operacional** do app `erp_axial_app`: como iniciar, quais rotinas estão disponíveis no menu, como trocar perfil de permissão e como apontar para um backend REST quando aplicável.

**Fora do escopo desta seção:** design do framework (ver documentação interna em `architecture/` do repositório), processo de migração Delphi → Flutter (playbooks internos) e rotinas ainda não publicadas.
