# CryptoPilot

Plataforma de automação de trading de criptomoedas que ingere sinais de múltiplas fontes, aplica controles de risco unificados e executa ordens automaticamente 24/7 via APIs de exchanges.

## Escopo

Este manual cobre:

- Instalação e configuração inicial do ambiente
- Registro e autenticação de usuário
- Criação, edição, ativação e exclusão de estratégias
- Gerenciamento de ativos com alocação de capital
- Configuração de parâmetros de risco por estratégia
- Monitoramento de whales e espelhamento automático
- Copy trading de top traders
- Análise de IA por ativo (entrada, stop, take profit)
- Backtesting de estratégias com dados históricos
- Configuração de notificações via Telegram e Discord
- Integração com alertas do TradingView via webhook
- Variáveis de ambiente e parâmetros operacionais
- Resolução de erros comuns

Não cobre:

- Desenvolvimento ou extensão do código-fonte
- Operações de infraestrutura avançada (Kubernetes, TLS personalizado)
- Integração com Quantfury (sem API pública disponível)
