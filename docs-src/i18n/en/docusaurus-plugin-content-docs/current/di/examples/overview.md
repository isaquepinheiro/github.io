## Repository examples

The repository ships Delphi examples under [`Exemples/Delphi/`](file:///d:/Ecossistema-Delphi/InjectContainer/Exemples/Delphi):

- `DependencyInjection_Classe`
- `DependencyInjection_Class_Lazy`
- `DependencyInjection_Interface`
