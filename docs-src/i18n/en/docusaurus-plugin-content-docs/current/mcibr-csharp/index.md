Biblioteca .NET (net8.0) com motor de cálculo tributário, organizada em torno de um pipeline: **ImpostoMotor → NotaFiscal → Produto → Impostos/RT → validações**.

Este projeto é uma biblioteca (não há `Program.cs`/API/console). O ponto de entrada de uso é a classe `ImpostoMotor`.
