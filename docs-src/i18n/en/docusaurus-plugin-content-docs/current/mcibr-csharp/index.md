A .NET library (net8.0) with a tax calculation engine, organized around a pipeline: **ImpostoMotor → NotaFiscal → Produto → Impostos/RT → validations**.

This project is a library (there is no `Program.cs`/API/console). The main entry point is the `ImpostoMotor` class.
